Dungeon Hub Bot

What is the Dungeon Hub Bot?

The Dungeon Hub Bot is a Discord bot that provides a variety of features to enhance the user experience on the Dungeon Hub Discord server.
Originally, the bot was created to keep track of the amount of services a user has provided for others on the server. Recently, more and more features were developed to aid with the entire administration of the server.

All features are described in detail in the subsections of this topic.
