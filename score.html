Dungeon Hub Help

/score ✏️

Description

Displays the current score breakdown for the selected user. If no user is provided, shows your own score.

Arguments

Name

Type

Description

Optional?

Additional

user

User

User to view the score of.

✅ Yes

Defaults to the issuer.

Examples

Your score

/score

Another user’s score

/score user: @user

See Also

  • /leaderboard: compare scores across users.

  • /log: log carries to earn score.

  • /manage-score: admin tools for adjusting score.
