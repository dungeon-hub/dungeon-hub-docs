Dungeon Hub Help

/link

Description

Links your account to a Minecraft player by IGN. If you are already linked, the bot tells you who you’re linked to and suggests /unlink.

Arguments

Name

Type

Description

Optional?

Additional

ign

String

Your Minecraft Ingame-Name.

❌ No

Min length: 2

Examples

Link by IGN

/link ign: Taubsie

Already linked

/link ign: Taubsie

See Also

  • /unlink: remove your link.

  • /sync, /force-sync ✏️ and /mass-sync ✏️: refresh roles after linking.

  • /send: post the link-message helper.

  • Verification ✏️: the verification feature is described closer here.
