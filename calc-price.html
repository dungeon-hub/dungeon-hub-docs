/calc-price

Examples

Description

This command calculates the price for some amount of carries. The price depends on the way the Carry Difficulty ✏️ is set up, as well as the amount of carries requested.

If the carry difficulty has a bulk price and amount configured, and the amount of carries is greater than or equal to the bulk amount, the bulk price is used. Otherwise, the price calculation uses the price of the carry difficulty.

The Type field consists of:
{carryTier.displayName} | {carryDifficulty.displayName}

The thumbnail url is set on the Carry Difficulty ✏️ as well, if there is none set, the thumbnail url of the Carry Tier ✏️ is used. If none is set there as well, there will be no thumbnail shown.
