/embed

Description

The embed command lets moderators work with Discord embeds directly. You can:

  • /embed get – extract the embed data from an existing message as a code block, a CDN link, or a readable preview.

  • /embed send – post one or more embeds from JSON into a channel (optionally using a CDN link as the source).

  • /embed add – append one or more embeds to a message sent by this bot (requires that the message already contains at least one embed).

  • /embed edit – replace a specific embed on a message sent by this bot.

See Embed Management ✏️ for JSON format details and tips, including CDN usage.
