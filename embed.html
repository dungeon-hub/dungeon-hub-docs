Dungeon Hub Help

/embed

Sub-Commands

  • /embed add ✏️: Add a new embed to a message by this bot.

  • /embed edit ✏️: Edit a specific embed on a message.

  • /embed get ✏️: Display all embed data of a given message.

  • /embed send ✏️: Send a new message containing the given embed(s).
