Dungeon Hub Help

/ign

Description

Looks up the IGN for the provided Discord user, if they are linked. If the user isn’t linked, an error is shown.

Arguments

Name

Type

Description

Optional?

user

User

Discord user to show the IGN for.

❌ No

Examples

/ign user: @taubsie

See Also

  • /link: link a Discord user to an IGN.

  • /find-user: inverse lookup by IGN.
