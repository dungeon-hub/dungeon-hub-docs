/config

Description

Read or modify server-level settings. Properties are typed (string, number, boolean, channel, category, role) and can be reset to null. See Server Property for available properties and meanings.

Sub-Commands

  • /config get: Show the current value for a property.

  • /config set: set string, set number, set boolean, set channel, set category, set role, as well as set null (which resets a property to null)
