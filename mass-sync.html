/mass-sync ✏️

Descripotion

This command allows you to queue many users for role/nickname syncing. This queue is server-based, where one user is synced every 15 seconds.

Sub-Commands

  • role: Add members of a role (or @everyone) to the sync queue.

  • guild: Add members of a Hypixel guild to the sync queue.

  • /mass-sync list: Show number of users currently queued.

  • /mass-sync clear: Clear the current queue.

See Also

  • /force-sync ✏️: single-user alternative.

  • /link: make sure users are linked before syncing.

  • Verification ✏️: the verification feature is described closer here.
