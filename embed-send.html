/embed send

Description

Sends a new message containing embeds. Provide JSON for a single embed or a JSON array for multiple embeds. You can also pass a CDN URL (generated by /embed get) instead of raw JSON.

If a CDN URL is provided, the bot downloads the JSON and posts the embeds to the selected channel.
