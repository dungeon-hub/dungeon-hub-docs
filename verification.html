Dungeon Hub Help

Verification ✏️

Verification links a Discord user to their Minecraft account (by them supplying their IGN (ingame name)) and keeps their roles and nickname in sync.

Linking and unlinking

To link your account, use /link. You can run it in DMs or in a server. The bot verifies your IGN and confirms if you have the same discord account as the one you set on Hypixel.
To remove the link later, use /unlink.

After linking or unlinking, roles and nickname are refreshed automatically where possible.

Admins can also prompt users with a convenient link UI by posting a helper message using /send link-message.

Syncing

Once a user is linked, their roles and nickname are kept in sync. This is explained further in Role Management

  • On demand, members can run /sync to apply the correct nickname and roles.

  • Staff can force a sync for another user using /force-sync ✏️.

  • For large updates, queue many users at once with /mass-sync ✏️ (by role or Hypixel guild).

    • This command also includes the /mass-sync list and /mass-sync clear utilities.

Quick lookups

  • Show the IGN for a Discord user with the /ign command.

  • Find the Discord user linked to a username with the /find-user command.

See also

  • /send for posting the link-message.

  • /lookup to check if a user/IGN is flagged, this also checks their linked account if one exists.
