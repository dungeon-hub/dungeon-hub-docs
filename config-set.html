/config set

Variants

  • string: set a string property.

  • number: set a numeric property.

  • boolean: set a boolean property.

  • channel: set a text channel property.

  • category: set a category channel property.

  • role: set a role property.

  • null: reset a property to no value.

Available properties and types: Server Property.
