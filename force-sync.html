/force-sync ✏️

Description

Recalculates and applies the correct roles and nickname for the selected user. If the user is not linked, only roles are synced.

Arguments

Name

Type

Description

Optional?

user

User

The target user.

❌ No

Examples

Force sync roles and nickname

/force-sync user: @User

See Also

  • /link and /unlink: linking affects role assignment.

  • /sync: this command effectively gets executed for another user if /force-sync is executed.

  • /mass-sync ✏️: queue large updates instead.

  • Verification ✏️: the verification feature is described closer here.
